# Tower Math Quest

`Tower Math Quest` is a browser prototype that merges:

- auto-attacking tower defense inspired by idle base-defense games
- primary-school math practice with Duolingo-style daily progression
- shield, boost, leveling, and coin-based upgrade loops

## How to run

Open [`index.html`](/Users/theunfrgiven/Documents/Playground/index.html) in any modern browser.

No build step is required.

## Current prototype features

- a base tower that attacks automatically
- three enemy types: fast scout, tank, and miniboss ogre
- shield absorption before base health takes damage
- an attack multiplier that grows with correct answers and upgrades
- four beginner math categories: addition, subtraction, multiplication, division
- XP leveling that improves attack, shield, health, and attack speed
- simple daily-goal framing with streaks, hearts, and stars
